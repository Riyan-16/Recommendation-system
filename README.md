# Recommendation System 

## 📌 Overview
This is a Python-based recommendation system that suggests movies and books based on user preferences using TF-IDF and Cosine Similarity.

## ✨ Features
- Recommends movies/books
- Interactive command-line interface
- Uses TF-IDF and Cosine Similarity
- Easy to use

## 🛠️ Technologies Used
- Python
- Pandas
- File handling (CSV Files)
- Scikit-learn

## 📂 Project Structure
project/
│── main.py
│── movies.csv
│── books.csv
│── recommendation.py
│── README.md

## 🚀 How to Run
1. Clone the repository.
2. Install the required libraries.
3. Run:
```bash
python main.py
```
